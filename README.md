# Flights Manager

Программа для управления данными об авиарейсах. Реализует:

- Добавление/удаление записей (номер рейса, авиакомпания, город вылета, город прилета, время в пути)
- Запрос 1: рейсы из заданного города
- Запрос 2: минимальное время прямого перелета между городами
- Запрос 3: города, обслуживаемые заданной авиакомпанией
- Запрос 4: гистограмма количества рейсов по авиакомпаниям

Среда: Qt Creator, C++, CMake
